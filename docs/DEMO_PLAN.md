# Demo Plan

## Goal

Show that Robo-T-Rader is a platform, not just a bot.

## Main demo surface

- `Prod-Hetzner`

Why:
it is the clearest operational face of the system and the best way to show multi-engine execution discipline.

## Supporting surfaces to show

- MarketData Hub
- Graphify
- Sim / MarketSim
- VFL
- Opportunity Coach
- Gem Mining
- Edge Discovery Engine
- EPIC Atlas

## Recommended demo narrative

1. Open with the ecosystem map
2. Show Prod-Hetzner as the main operational runtime
3. Show Hub as the shared market data plane
4. Show Sim / VFL / Opportunity Coach / Gem Mining as the learning workflow
5. Show Edge Discovery / Incubator / Atlas as the future edge pipeline
6. Close with Graphify and governance as the system-wide control layer

## Key message

The project is valuable because it makes a technical team more effective at building and governing autonomous trading systems.
